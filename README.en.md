# Skill Learning

This repository collects Cursor / Agent skills, harness-oriented workflows, and practical AI delivery methods.

The goal is not just to save prompts. The goal is to turn real project practices into reusable assets:

- project intake
- problem framing
- first-principles analysis
- task decomposition
- skill / harness distillation
- rollout and release materials

## Current Skills

- `engineering-problem-gate`
  - A project-grade problem gate for engineering teams
  - Used to collect project context, frame the problem, decompose the work, and decide whether something should become a `skill` or `harness`

- `framing-problems-before-building`
  - A lightweight skill for clarifying vague problems before design or implementation

- `generating-skill-definitions`
  - A helper skill for drafting and iterating `SKILL.md` files

- `learning-summary`
  - A note-to-summary skill for organizing learning materials

## Who This Repository Is For

- engineering teams that want to bring AI usage into project workflows
- teams that want to frame problems before execution
- teams that want to distill internal skills and harnesses
- practitioners building Cursor / Agent delivery methods

## Usage

Place the needed skill directory in:

```text
.cursor/skills/
```

or in your personal skills directory:

```text
~/.cursor/skills/
```

## Recommended First Skill

If you want to try only one skill first, start with:

```text
engineering-problem-gate
```

It is the closest to real project work and works well as a common team entry point.
