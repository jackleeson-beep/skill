# Project Intake

Use this template before applying the gate to a real project.

Its job is to make sure AI has enough context before anyone asks it to design, decompose, or implement.

## Minimum Context to Collect

- project goal
- current pain point
- target users or customers
- available resources
- hard constraints
- success criteria
- what AI can and cannot access

## Rule of Thumb

If this context is missing, do not jump into:

- detailed solutioning
- formal skill creation
- formal harness design
- broad AI execution
